# Fleet Management Dashboard

A responsive fleet management web dashboard built with HTML, Tailwind CSS, and JavaScript.

## Features

- Vehicle list with registration number, make, model, and status
- Individual vehicle profile modal with expenses, damage history, service history, documents, and tyre replacements
- New claim/expense submission form with invoice and photo uploads
- Expense tracking dashboard with totals, monthly trend and approval workflow status
- Search and filter controls for registration, expense type, date, and approval status
- Mobile-friendly responsive layout
- Sample dummy data included for demonstration

## Run locally

1. Open `index.html` in your browser.
2. Use the dashboard to explore vehicles, open profiles, and submit claims.
